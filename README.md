# Saiint-Beauty
A modern full-stack web application designed to simplify appointment scheduling and customer loyalty management for beauty salons. The system provides customers with a seamless booking experience while giving salon administrators the tools to manage appointments, services, and customer rewards efficiently.

## OverView
## Features
## Customer Features
- Create an account and securely log in
- Browse available beauty services
- Book appointments online
- View upcoming appointments
- View appointment history
- Earn loyalty points after completed appointments
- Redeem loyalty rewards
  
## Administrator Features
- Secure administrator login
- Manage customer accounts
- Manage salon services
- View and manage appointments
- Change availability
- Approve or update bookings
- Award and manage loyalty points
